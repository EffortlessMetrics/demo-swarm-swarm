## 2024-10-18 - [Redundant Coding in Text-Based UIs]
**Learning:** Even in text-only interfaces (like Markdown reports), accessibility principles apply. Using emojis as icons alongside text labels serves the same purpose as redundant coding in GUIs (e.g., color + shape).
**Action:** When designing agent outputs, always instruct the agent to pair status text with semantic emojis to improve scanability and accessibility for color-blind users.
