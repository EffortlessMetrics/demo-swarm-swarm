# Tutorials

Learning-oriented: follow the steps to completion with the canonical goal (`demoswarm version` CLI subcommand).

| Doc | Use it when | Outcome |
|-----|-------------|---------|
| [Quickstart](quickstart.md) | New to the pack and want Flow 1 working fast | `.runs/<id>/signal/*` written; Flow 1 receipts verified |
| [Toy Run](toy-run.md) | Want to see Flow 1 artifacts without GitHub | Offline-friendly `.runs/<id>/signal/*` artifacts you can inspect |
| [Validation Run](validation-run.md) | Need to validate pack behavior (Flows 1-4) in a sandbox | Receipts for Flows 1-4 plus optional GitHub issue/updates |
| [Walkthrough](walkthrough.md) | Presenting/demonstrating the pack end-to-end | Narrative demo of the canonical goal with artifacts to show |
